Understand "look" as looking.
Examining is an action applying to a thing.
Understand "examine" as examining.
Understand "x" as examining.

Going is an action applying to something.
The going action has an object called the room gone from.
The going action has an object called the room gone to.
The going action has an object called the door gone through.
Understand "go" as going.
Check going direction:
    if location has no connection in direction, instead say "Nowhere to go".

When play begins, try going location.
When report going somewhere:
    say "[printed name of location]\n"
    say "[description of location]\n\n"
